##他人读书经验
基础版：股票就是交易

1、总体市场的大趋势决定个股的小趋势，股价往阻力最小的方向前进；

2、报价上的执行偏差，如何及时的买入和卖出。如A股的T+1制度，网速问题；

3、希望（贪婪）和恐惧是人的本性，很难割除。交易者之路是一条认清自我、超越自我之路。诚实地面对自我，面对市场；

4、脸有镜子照，交易行为没有镜子。市场就是一面镜子，必须正视市场；

5、敬业、守纪律是市场参与者的根本立足点；

6、股票的本质就是交易，只要每一笔有利润就行；

7、根据趋势来进行交易，成交量表示股票的活跃程度。对于成交量良好、上升趋势确立的股票，就是最佳的买点，这个时候获得交易利润的可能性最大；（2014，A股是羊群市，散户跟风强烈，市场不成熟，因而目前A股是利用趋势+量能赚钱的最好时期）

8、技术要领——只有看到市场前兆对交易有利时，才进入市场交易。不顾实际市场情况，只顾不停地买卖，是许多交易亏损的罪魁祸首；

9、趋势——必须确认趋势成立，必须确认当前市场环境有利于交易。两者如有不符，需要等待。即便在等待之中股价上涨也不在乎，因为等待中确认了趋势的成立，减小了亏损的概率；

10、市场是不会错的，错的是我们自己，他不是多头的方面，也不是空头的方面；

11、你必须相信自己，相信自己的判断，如果你想在这一行谋生的话；

12、投机是一项艰苦的行当，必须全力投入。唯一证明自己错误的就是赔钱，只有自己赚钱了才是正确的。因为，这是交易，无需寻找理由；

13、认为应当卖出时就卖出，当觉得股票要上涨时就买进。这就是投机的基本原则！

14、股市上涨过程中，总有人会站出来说这样那样的问题，认为除了他们自己之外，全世界的人都疯了，但是除了他们自己之外，所有人都赚钱了。这就是趋势的力量。

15、牛市上涨中，短线交易是得不偿失的，因为会错过上涨周期。

16、市场下跌终有尽头，股票也不会涨到天上去。然而，在没有任何征兆转势之前，逆势而为是非常危险的。抄底是神作的事情。

17、上涨和下跌都有趋势，我们可以在上涨的趋势中获利，同样可以在下跌的趋势中赚钱。（开通融券势在必行啊）

升级版：最大限度获取交易利润

18、参考市场当前状况，按照自己的系统交易。

19、没有判断，就没有利润。在大势确立之时，应该耐心持股，让利润奔跑。

20、赌博与投机具有本质区别。来股市就是投机，就是尽可能赚取更多的利润。

21、不能让自己的状态影响交易，不能让自己精神涣散和四肢麻木地参与交易，因此好的休息和睡眠是必须的。在交易间隙，如周末、节假日，应该尽情去玩。

22、真正的高手，少说多听。凡是能帮助自己更明智去交易的，来者不拒，从不厌倦。

23、股市中的肥羊：1)初学者，一无所知，牛市能赚去最大利润，熊市则亏光本金，不懂得止损；2）有经验的肥羊（大多数雪友），打听到了更高阶段的肥羊（雪球大V），有一些避免金钱损失的方法，常常不顾大势，频繁交易，从而慢慢亏光本金或收获很少利润；3）真正的高手，能判断大势，能耐心持有(是否高手的区别)。众多高手才俊，不能避免亏损，原因是耐心所致。

24、在牛市行情中，做法只能是买进并耐心持有，直到确信牛市已经接近尾声。而要判断是否牛市尾声，需要研究总体市场状况，而不是个股。（20140102，新股IPO和注册制-->创业板已是牛市尾声)

25、神奇的直觉来自交易灵感。如下跌市中上涨的股票需要关注以便行情好时做多，上涨市中下跌的股票需要留意以便熊市来临是做空。

26、趋势中，股价走势有时候反映了底牌。底牌不揭晓，行情不结束。（茅台03）

顺势步步为营建仓：

27、可以告诉别人看空还是看多，但不能告诉他人买进或卖出任何具体的股票。因为，询问的人很懒，想不劳而获。

28、琢磨个别股票比研究总体市场更轻松，研究市场的运动比研究个别股票的波动更重要。在正处于上升状态的市场中买进，是最舒服的股票买入方法。

29、不要按照步步下降的方式买入做多，而应该按步步上涨的方式买入做多。关键点不在于每次买入最低和卖出最高，而应一定在买和卖在正确的时机。

30、假定我们已经研究了总体市场状况，并且大势看涨，也应该步步为营的方式建仓。不要希望买进股票的价格太低廉，或者到手太容易。不愿意低价买进股票的缘故？？？P88

31、股票永远不会因为价格太高而不可买进，或者因为价格太低而不可卖出。当你第一笔交易买入后，除非有利润，否则就不要做第二笔。第一笔买入亏损，说明错了，最起码暂时错了。做空也是一样，首先是建立空头头寸，正确了再继续做。

买入和卖出时机的重要性：（错误的时机可能导致“我判断正确，却破产了”）

32、相信自己，杜绝小道消息。如东方集团，应该通过观察股东情况、基本面、行业、量能和趋势等确定股票的运行方向。）

33、不要让屁股决定自己的脑袋。很多精明的人看多或看空，是因为他们本身自己已经买入或卖出股票。应该理性去看待市场，忘记自己的股票。

34、如果市场意外地让、甚至毫无道理地让你不利，对自己生气相当于肺炎患者对自己的肺生气。不要和市场生气，一定是自己什么地方错了。

35、买入之前，一切的头等大事是要先确定当前市场大势，而大势是由基础条件环境所决定的。大规模行情一旦来了，无论谁来阻碍，行情都将不可避免地按照其推动力量所决定的幅度、速度和持续时间来展开。（行情何时逆转？）

36、该做的是“在牛市行情看多、熊市行情看空。”正道是就是从日常价格波动回归到市场基本趋势。

37、短线炒作小幅波动立竿见影，长线买入则获利更丰但是来得比较慢。（对于本金多者，应当适当布置长线，避免市场波动，获取厚利）

38、有了足够的资源（本金），也有了足够的经验和信心，那么就得关注时机之锁。（即便判断对了方向，也应注意介入时机，否则可能倒在黎明之前）

39、根据基础条件对市场进行总体判断，在判断正确的情况下也应等待出手时机，且需要缓慢出手。如果出手过快，则可能会倒在黎明到来之前。（金字塔建仓的重要性）

40、在熊市行情刚刚开始的时候即使看空是完全正确的，最好也不要马上开始大笔卖出，一定要等到发动机不存在回火风险之后。极尽所能地仔细观察并耐心等待，等待出手时机。

41、在一系列市场事件之中存在着不可抗拒的内在逻辑，利用它帮助我们获利。关键是选择正确的一边，知道哪边正确，并采取相应措施。

42、卖出时机在于“判断下跌或上涨的幅度已经贴现了近期市场前景。预期即将出现相当幅度的反弹，但是还不至于看好到入市做反转行情”。（卖出时机：还是老的预期已经实现，新的预期已经熊熊燃烧起来）

从基本形式预见流动性危机成为市场之王：

市场最小阻力线（初现的关键时刻）：

58、股票投机者通常犯这样那样的错误，并且常常明知故犯。明智的投机者不会再次犯相同的任务，然而错误家族如此庞大，当你想试试身手的时候，总有一个成员在你身边打转。

62、需要判定的关键因素是：从交易的那一刻向前展望的阻力最小行情路线。然后顺势买入（小头寸），在上涨中继续买入（扩大头寸），逐步设定止损线。

63、在箱体行情中，价格的来回波动有助于最小阻力线的形成。在此过程中有空翻多，多翻空。属于横盘过顶买入，横盘时间1年以上为好。

64、在牛市行情下，看空的消息被市场忽略，看多的消息被市场扩大。反之亦然。

65、随时做好准备，一旦判明市场阻力最小的路线便立即顺其方向交易。正确的一边总是得到两股力量的帮助，一是基本形式（形成趋势），一是那些站在错误一边的人（扩大趋势）。

66、在窄幅波动的市场行情中，当价格谈不上任何明确方向而只是在狭窄范围内航向延伸时，企图预期市场的方向是没有任何道理的。

67、投机客必须一心一意地顺从市场才能获取利润，均不能执迷不悟地要求价格走势顺从自己。

68、愿意凭着对获取巨大利润的向往而盲目赌博呢，还是愿意明智地投机，取得数额较小但是可能性大得多的利润呢？（对于震荡市场行情，选择明智地投机，而不是抄底）

69、在正确的时间站在正确的一边，企图建立起有利可图的头寸，重仓赢取利润。（最小阻力线突破买入或卖出，建立头寸，然后继续加大仓位，并设立相应的止损线）
